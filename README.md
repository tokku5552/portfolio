## Getting Started

package install

```bash:
yarn install
```

prepare for local

```bash:
cp -pr .env.sample .env
# please override environment variables
```

start local server

```bash:
yarn dev
```

- testing

```bash
# lint
yarn lint

# unit test
yarn test
```
